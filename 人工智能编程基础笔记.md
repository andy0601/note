## 变量赋值、运算符
- 变量名用小写和_组成
- 运算符
```python
print(3 ** 2)               # 取幂
print(10 // 3)              # 向下取整
print(10 % 3)               # 取余
print(float(7))             # 7.0
print(type(4))              # int
print(type(7.0))            # float
print('hello' * 3)          # hellohellohello
print(len('udacity'))       # 7
print('hello'[0])           # h
print(len(835))             # Error
print('red fish, green fish'.count('fish'))     # 2
print('red fish, green fish'.islower())         # true
```
